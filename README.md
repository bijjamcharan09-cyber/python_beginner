# 🐍 Python Beginner Projects

A collection of beginner-friendly Python projects created while learning Python programming. These projects focus on strengthening programming fundamentals such as variables, functions, loops, conditions, file handling, and user input.

## 📂 Projects

### 1. 🧮 Calculator (`calculator.py`)
A simple command-line calculator.

**Features**
- Addition
- Subtraction
- Multiplication
- Division
- User-friendly menu
- Handles multiple calculations

**Concepts Used**
- Functions
- Conditional statements
- Loops
- User input

---

### 2. 💰 Simple & Compound Interest Calculator (`ci_si.py`)
Calculates Simple Interest (SI) and Compound Interest (CI).

**Features**
- Simple Interest calculation
- Compound Interest calculation
- User input for principal, rate, and time

**Concepts Used**
- Mathematical formulas
- Variables
- Functions
- User input

---

### 3. 📊 Expense Tracker (`expenses.py`)
A basic expense tracker that stores expenses in a text file.

**Features**
- Add new expenses
- View all expenses
- Calculate total expenses
- File handling for data storage

**Concepts Used**
- File handling
- Lists
- Dictionaries
- Functions
- Exception handling

---

### 4. 🎓 Student Manager (`student_manager.py`)
A simple student record management system.

**Features**
- Add student details
- View student records
- Search student details
- Update student records
- Delete student records
- Save records using file handling

**Concepts Used**
- CRUD operations
- File handling
- Functions
- Lists
- Dictionaries

---

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/bijjamcharan09-cyber/python_beginner.git
```

2. Open the project folder

```bash
cd python_beginner
```

3. Run any project

```bash
python calculator.py
```

Example:

```bash
python ci_si.py
python expenses.py
python student_manager.py
```
